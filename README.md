# AnalisadorLexico

Pontifícia Universidade Católica do Paraná - PUCPR
Construção de Interpretadores - Professor Frank Alcantara
Aluna: Bell Jamyle Duwe Klemann

O projeto consiste em criar um analisador léxico em que a entrada são expressões em escrita RPN e a saída é um código Assembly das expressões. O projeto foi feito em Python.
O projeto deve ser executado da seguinte forma em linha de comando "python3 main.py test_expressions.txt".
